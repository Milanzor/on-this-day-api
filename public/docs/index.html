Introduction

Base URL: https://on-this-day-api.helopsokken.nl

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Fetch events that happened on a specific day

Example request:
curl --request GET \
    --get "https://on-this-day-api.helopsokken.nl/api/v1/events/that-happened-on/12/31?limit=10&category=births&language=en" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "https://on-this-day-api.helopsokken.nl/api/v1/events/that-happened-on/12/31"
);

const params = {
    "limit": "10",
    "category": "births",
    "language": "en",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Request      

GET api/v1/events/that-happened-on/{month}/{day}

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

URL Parameters

month   string   

The month of the event Example: 12

day   string   

The day of the event Example: 31

Query Parameters

limit   integer  optional  

The limit of the events Example: 10

category   string  optional  

The category of the event Example: births

Must be one of:
  • births
  • deaths
  • holidays
  • selected
  • other
language   string  optional  

The language of the event Example: en

Must be one of:
  • en
