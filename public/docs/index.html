Authenticating requests

This API is not authenticated.

Endpoints

GET api/v1/events/that-happened-on/{month}/{day}

Example request:
curl --request GET \
    --get "http://localhost/api/v1/events/that-happened-on/reprehenderit/est" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/v1/events/that-happened-on/reprehenderit/est"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
